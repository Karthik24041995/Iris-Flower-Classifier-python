# Iris-Flower-Classifier-python
# Iris Flower Classification

This is a simple machine learning project that uses the [Iris dataset](https://scikit-learn.org/stable/datasets/toy_dataset.html#iris-dataset) to classify flowers into three species using a Random Forest Classifier.

## Dataset

The Iris dataset is a classic multiclass classification problem, containing 150 samples of iris flowers with the following features:

- Sepal length
- Sepal width
- Petal length
- Petal width

## Project Structure

- `iris_classifier.ipynb`: Main Jupyter notebook for training and testing the model.
- `requirements.txt`: List of dependencies.

## Running the Project

> You need Python 3.x to run this project.

1. Install dependencies:

   ```
   pip install -r requirements.txt
   ```

2. Open the notebook:

   ```
   jupyter notebook iris_classifier.ipynb
   ```

3. Run all cells to train the model and test predictions.
